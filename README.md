# Farmer Information Portal - Farmer's Friend

An academic web application developed using PHP and MySQL to manage and display agricultural crop information and government schemes.


## Project Overview

The Farmer Information Portal is a simple full-stack academic project focused on understanding backend development fundamentals.  
It provides a public interface to view information and an admin panel to manage crops data and schemes data.


## Features

- User-friendly and visually rich UI designed using Figma
- Admin login using PHP sessions
- Admin dashboard to navigate 
- Add, view, and delete crop information  
- Add, view, and delete scheme details  
- Contact Us feature to collect user queries and store them in the database  
- Clean and modular project structure


## Technologies Used

- HTML  
- CSS
- Figma (UI design)
- PHP  
- MySQL  


## Project Type

Academic / Learning Project (MCA)


## How It Works

- Public users can view crop and scheme information.
- Users can submit queries using the Contact Us form.
- Admin logs in to manage crops and schemes using CRUD operations.
- Contact messages are stored in the database for admin review.


## Setup Instructions

1. Install **XAMPP**
2. Place the project folder inside `htdocs`
3. Start **Apache** and **MySQL**
4. Import the database using **phpMyAdmin**
5. Open browser and visit: http://localhost/Farmer_Information_Portal
